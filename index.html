Intro

This is super easy way to bring HTML5 audio player to your Foundation based project. Take a look to README and the source of this page. This tab is also sandbox for Foundation Player API. There is the player global variable. Fill free to experiment in your browser's JavaScript console.

playPause() This method toggles play or pause state of player. player object
play() toggles play state of player. player object

Look and feel
Method Description Returns
togglePlayerSize() This method toggles player size. Method returns current size of player which was set. 'small' | 'normal'
setPlayerSize(size) This method sets player size to one of possible values: 'small' or 'normal' . Player has 'normal' size by default. player object
